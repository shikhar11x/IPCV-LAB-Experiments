## Experiments

| File | Experiment |
|------|------------|
| `exp_1.py` | **Image Acquisition & Display** — Acquire and display an image; demonstrate image sensing, acquisition process, and components of an image processing system |
| `exp_2.py` | **Sampling & Quantization** — Perform sampling and quantization on an image; visualize effects of different sampling rates and quantization levels on image quality |
| `exp_3.py` | **Spatial Domain Enhancement** — Implement contrast stretching and histogram equalization; compare results before and after enhancement |
| `exp_4.py` | **Frequency Domain Filtering** — Apply low pass and high pass filters using FFT; demonstrate their effects on image |
| `exp_5.py` | **Noise Modeling & Spatial Filtering** — Simulate Gaussian and salt-and-pepper noise; apply mean and median filters for image restoration |
| `exp_6.py` | **Periodic Noise Reduction** — Reduce periodic noise using inverse filtering and Wiener filtering |
| `exp_7.py` | **Geometric Transformations** — Implement rotation, scaling, and translation on images |
| `exp_8.py` | **Color Image Processing & Segmentation** — Process color images and perform color-based segmentation |
| `exp_9.py` | **Image Compression** — Implement Huffman Coding and Run Length Coding for image compression |
| `exp_10.py` | **Edge Detection & Segmentation** — Detect edges using Sobel, Canny operators and perform image segmentation |
| `exp_11.py` | **Morphological Operations** — Apply dilation, erosion, opening, and closing morphological operations |
| `exp_12.py` | **Threshold & Region-Based Segmentation** — Segment images using global thresholding, Otsu's method, and region-based techniques |
| `exp_13.py` | **Shape Representation & Chain Code** — Represent object shapes and compute chain codes for boundary description |
| `exp_14.py` | **CNN for Image Classification** — Build and train a Convolutional Neural Network for image classification |
| `exp_15.py` | **Motion Estimation & Object Tracking** — Estimate motion and track objects across video frames |
| `exp_16.py` | **Face Recognition using Machine Learning** — Implement face recognition using machine learning techniques |

